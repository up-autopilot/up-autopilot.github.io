Welcome to the Raspilot

The Raspilot is a modular autopilot systems for the RC models intented for use on the Raspberry Pi.

Purpose

The Raspilot has been created as a proof of concept of the Up framework.

It has been created as a part of the Master's Thesis on the Charles University.

The Idea

The main idea of the Raspilot is proove the ability of the Up to integrate data sources in order to be able to control the aircraft and provide reliable telemetry source.

It is a system, which can handle the following use cases:

  • transmit telemetry to the ground station
  • stabilize the aircraft
  • communication with the ground station

The Required Peripherals

The Raspilot requires several peripherals in addition to the Raspberry Pi:

  • Arduino - Use Raspilot for Arduino
    • reads RX and IMU, controls the airplane's attitude
  • Android - Use Raspilot for Android
    • reads various sensors, provides GPS and Internet connection
  • Proxy Server - Use Up Proxy Application
    • exchanges data between Raspberry Pi and ground station, must be run on a machine with public IP address
