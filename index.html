Welcome to the Up framework

The Up Framework is a modular framework intended to ease creation of the autopilot systems for the RC models.

Purpose

The Up Framework has been created to address the following problem: There are many superb autopilot systems for RC models, but many of them lacks the ability to be modified and extendend easily.

It has been created as a part of the Master's Thesis on the Charles University.

The Idea

The main idea of the Up is to integrate various data sources into one place where they can be later used for various purposes such as aircraft control, telemetry, etc.

The Key Attributes

The key attributes of the Up are:

  • modularity
  • extensibility
  • modifiability
